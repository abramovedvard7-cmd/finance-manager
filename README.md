# Book Tracker — Приложение для учета прочитанных книг

Итоговый проект по курсу разработки на Python. Приложение позволяет пользователю вести список прочитанных книг, фильтровать их и сохранять данные в надежном формате.

## Автор
Петреченко Ангелина

## Описание программы
«Book Tracker» — это графическое приложение (GUI), созданное на библиотеке Tkinter. Оно предназначено для любителей чтения, которые хотят систематизировать свою домашнюю библиотеку или список изученной литературы.

### Основные функции:
•   **Добавление книг:** сохранение названия, автора, жанра и количества страниц.
•   **Валидация:** проверка на пустые поля и корректность ввода страниц (только числа).
•   **Фильтрация:** возможность быстро найти книги определенного жанра или книги объемом более 200 страниц.
•   **Хранение данных:** автоматическое сохранение и загрузка данных из файла `books.json`.
•   **Интерактивность:** получение случайных мотивирующих цитат о чтении через встроенный модуль (имитация API).
