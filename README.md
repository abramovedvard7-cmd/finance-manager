# Expense Manager (GUI)

Простое приложение на Python для учета личных расходов.

## Функционал
- Добавление расходов с указанием названия, суммы и категории.
- Просмотр истории в виде таблицы (Treeview).
- Удаление записей.
- Автоматическое сохранение данных в формат JSON.
- Валидация ввода (защита от пустых строк и некорректных чисел).

## Требования
- Python 3.x
- Библиотека Tkinter (встроена в стандартную библиотеку Python)
